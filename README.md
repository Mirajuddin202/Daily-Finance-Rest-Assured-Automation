# Daily Finance Management System – API Test Automation (Rest Assured)

## 📌 Project Overview

This project is an **API Test Automation Framework** for the **Daily Finance Management System**, built using **Java**, **Rest Assured**, and **TestNG**. The framework follows the **Page Object Model (POM)** architecture to ensure clean, reusable and maintainable test code.

It covers both **positive** and **negative** test scenarios for **User** and **Item Management APIs**.

---

## 🚀 Features

### 👤 User Management
- ✅ Register a new user
- ✅ Admin login
- ✅ User login
- ✅ Retrieve user list
- ✅ Search user by ID
- ✅ Update user information (First Name, Phone Number)

### 📦 Item Management
- ✅ Get item list
- ✅ Add a new item
- ✅ Update an existing item
- ✅ Delete an item

---

## ✅ Test Coverage

### Positive Test Cases
- User registration
- Admin login
- User login
- Retrieve user list
- Search user by ID
- Update user information
- Add item
- Update item
- Delete item

### Negative Test Cases
- Admin login with invalid credentials
- User login with invalid credentials
- Register with existing email or phone number
